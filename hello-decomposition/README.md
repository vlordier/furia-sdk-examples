# hello-decomposition

Demonstrates the `DecompositionStrategy` SDK trait — splitting a mission into F2T2EA sub-missions.

## What it shows

- Implementing `DecompositionStrategy` for mission decomposition
- Generating Find, Fix, Track, Target, Engage, Assess phases
- Scoring Courses of Action
- Health/version via `ModuleHandle`

## Run

```bash
cargo run -p hello-decomposition
```
