# Furia SDK Examples вҖ” Architecture

## Overview

This repo contains 25 example crates, each demonstrating one Furia SDK trait or
shared type family. They serve as copy-paste starting points for external C2
plugin developers.

## Example Structure

Each hello-* crate:
- Defines a struct implementing one SDK trait
- Registers via `FuriaBuilder::with_provider()`
- Exposes health/version via `ModuleHandle`
- Runs as a standalone binary or test
