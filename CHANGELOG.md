# Changelog

## 0.1.0 (2026-07-03)

Initial release — 18 example crates demonstrating core Furia SDK traits and shared module patterns.

### Added
- hello-acoustic through hello-ui: 18 standalone demos covering AcousticProvider, AssessmentEngine, CbrnProvider, CivilianDensityProvider, DecisionTreeProvider, DecompositionStrategy, DispatchAdapter, EWSimProvider, ExportAdapter, FusionEngine, IntentProvider, LogisticsProvider, PlatformProvider, PolicyProvider, SensorAdapter, SimulationProvider, TerrainAnalyst, UiPlugin, plus a NATO coalition lifecycle/audit demo using SDK security context types
- Each crate is a standalone trait or shared-type demonstration with local demo structs and tests
- Roadmap notes for planned C-UAS trait examples: AirspaceManager, EngagementPlanner, InterceptorPairingProvider, KillChainOrchestrator, ThreatScorer
- README with example catalog table
